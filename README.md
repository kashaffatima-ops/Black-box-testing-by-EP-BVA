# Online Event Registration System - Test Design Project

## Overview

This repository contains a comprehensive Software Quality Assurance (SQA) project focused on the testing and validation of an Online Event Registration System. The project applies industry-standard Black-Box Testing techniques including Equivalence Partitioning (EP) and Boundary Value Analysis (BVA) to design effective and efficient test cases.

## Objectives

* Identify valid and invalid input conditions.
* Apply Equivalence Partitioning for input classification.
* Apply Boundary Value Analysis for edge-case testing.
* Design comprehensive test cases for system validation.
* Improve software reliability through structured test design.

## Testing Techniques Used

### Equivalence Partitioning (EP)

Input data is divided into valid and invalid partitions to reduce the total number of test cases while maintaining effective coverage.

### Boundary Value Analysis (BVA)

Focuses on testing boundary conditions where defects commonly occur, such as minimum, maximum, and out-of-range values.
